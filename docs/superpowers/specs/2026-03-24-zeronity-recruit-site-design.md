# Zeronity 採用サイト - 設計仕様書

## 概要

Zeronity株式会社の採用専用サイトを、既存コーポレートサイト（WordPress / 0-i.co.jp）とは独立したNext.jsプロジェクトとして新規構築する。

「スタートアップファーム」という独自コンセプトを強く打ち出し、新卒・中途・フリーコンサル/業務委託の全ターゲットに対して、Indeed・engageへの応募誘導をゴールとする。

## 背景と目的

### 背景
- Zeronity株式会社はコンサルティング事業を中心に、人材教育・環境開発の3事業を展開するスタートアップファーム（従業員67名）
- 現在のコーポレートサイト（WordPress）には採用ページが存在しない
- 採用資料（PDF 42ページ）は存在するが、Web上での採用導線がない

### 目的
- 採用ターゲット（新卒・中途・フリーコンサル/業務委託）に対してZeronityの魅力を伝え、Indeed・engageでの応募につなげる
- 「スタートアップファーム」という独自ポジションを強くブランディングする

### 成功基準
- 採用サイト経由でIndeed・engageへの遷移が発生すること
- 会社の理念・事業内容・働く環境が求職者に伝わること

## 技術スタック

| 要素 | 選定 | 理由 |
|------|------|------|
| フレームワーク | Next.js (App Router) | 既存プロジェクトの知見あり、SSGで高速表示、リッチなアニメーション実装が容易 |
| 言語 | TypeScript | 型安全性 |
| スタイリング | Tailwind CSS | ユーティリティファーストで高速開発、ダークテーマとの相性が良い |
| アニメーション | Framer Motion | スクロールアニメーション、ページ遷移 |
| デプロイ | Vercel | Next.jsとの親和性、無料枠で十分 |
| ドメイン | recruit.0-i.co.jp（サブドメイン） | コーポレートサイトのドメインパワーを活用 |

### 除外した選択肢
- **WordPress追加ページ**: テーマの制約でリッチな演出が困難
- **Astro**: リッチなインタラクションの実装にNext.jsより手間がかかる
- **Headless CMS**: コンテンツの更新頻度が低いため不要

## デザイン仕様

### カラーパレット

| 用途 | カラー | 備考 |
|------|--------|------|
| 背景（メイン） | `#0a0a0a` | ダークベース |
| 背景（グラデーション） | `#1a0a2e` → `#1a0508` | 紫→赤の暗めグラデーション |
| アクセント（紫） | `#7a00df` | コーポレートブランドカラー |
| アクセント（赤） | `#e8380d` | コンサル事業カラー |
| アクセント（グラデーション） | `#7a00df` → `#e8380d` | 採用サイト独自のアイデンティティ |
| テキスト（メイン） | `#ffffff` | 白 |
| テキスト（サブ） | `rgba(255,255,255,0.5)` | 半透明白 |
| テキスト（ラベル） | `rgba(255,255,255,0.4)` | 薄い白 |

### デザイントーン
- 黒背景 × 紫→赤グラデーション × 白テキスト
- コーポレートサイト（0-i.co.jp）のトーンを踏襲しつつ、採用サイト独自のグラデーションで差別化
- 幾何学モチーフ（三角形、六角形、球体）をアクセントとして使用
- コーポレートサイトのAOSアニメーション（スクロール連動）を踏襲

### タイポグラフィ
- 日本語: Noto Sans JP（またはコーポレートサイトと同一フォント）
- 英語: Inter または Montserrat（見出し用）
- セクションラベル: 英字大文字 + letter-spacing広め

### 写真素材
- 初期はダミー/ストック写真で構築
- 後から実際の社員写真・オフィス写真に差し替え予定

**目的:** 応募に必要な情報を提供し、Indeed・engageへ誘導する

**セクション構成:**

#### 7-1. 募集職種

#### 7-2. 選考フロー
- タブ切り替え: 新卒 / キャリア / フリーコンサル

- **キャリア**: 書類選考 → カジュアル面談 → プレゼンテーション面接 → 最終面接 → オファー面談
- **フリーコンサル**: カジュアル面談 → スキルシート確認 → 条件面談 → 契約（詳細は問い合わせ導線を設置）
- ステップ形式のビジュアル

#### 7-3. FAQ
- アコーディオン形式
  - どのような経歴の方が在籍していますか？
  - プロジェクトにはどのようにアサインされますか？
  - 副業は可能ですか？
  - 就業時の服装の指定はありますか？
  - オンライン面接は可能ですか？
  - どんなスキル・能力を重視しますか？
  - プレゼンテーション面接とは何ですか？
  - インターンシップではどのようなことが経験できますか？

#### 7-4. CTA
- 大きな「応募する」セクション
- Indeed・engageの2ボタンを目立つ配置で表示

- **デスクトップ**: 1200px以上（メインターゲット）

- **モバイル**: 〜767px

- サブドメイン（recruit.0-i.co.jp）でコーポレートドメインとの関連性を維持
